# Scripts

Bu klasör, proje için yardımcı script'leri içerir.

## `banners/`
Banner görselleri oluşturmak için kullanılan Matplotlib tabanlı Python script'leri.

```bash
# Aktif banner'ı yeniden oluşturmak için:
python scripts/banners/create_ultimate_teknofest_banner.py
```

## `demo_dsp.py`
Sinyal işleme özelliklerinin interaktif demonstrasyonu.

```bash
python scripts/demo_dsp.py
```
